# 04. 연차 저금통 (Annual Leave Calculator Spec)

> 출처: `packages/shared_logic/lib/src/utils/payroll/annual_leave_calculator.dart`

---

## 1. 연차 발생 규칙

### 1.1 1년 미만 근로자 (월별 연차)

```
입사일로부터 1개월 만근 시 1개의 유급휴가 발생
최대 11개 (12개월 미만 중 만근한 월 수)

판정 기준:
  FOR month = 0 TO 11:
      monthStart = 입사일 + month개월
      monthEnd   = 입사일 + (month+1)개월 - 1일
      
      IF monthEnd > 정산시점 → BREAK (아직 도래하지 않은 월)
      
      IF 해당 월 만근:
          → +1개 연차 발생
      ELSE IF 해당 월 근태 기록 없음 (앱 도입 이전):
          → 만근으로 간주 (+1개)

법적 근거: 근로기준법 제60조 제2항
```

### 1.2 1년 이상 근로자 (정기 연차)

```
1년 주기 연차: 80% 이상 출근 시 15개 유급휴가

판정 기준:
  FOR year = 1 TO N:
      yearStart = 입사일 + (year-1)년
      yearEnd   = 입사일 + year년 - 1일
      
      expectedDays = 해당 기간 내 계약 근무일 수
      workedDays   = 해당 기간 내 실제 출근일 수
      
      출근율 = workedDays / expectedDays
      
      IF 출근율 ≥ 80%:
          기본 = 15개
          추가 = (year - 1) ÷ 2  (2년마다 +1개)
          연차 = min(기본 + 추가, 25)  ← 상한 25개
      ELSE:
          연차 = 0개 (미발생)

법적 근거: 근로기준법 제60조 제1항, 제4항
```

### 1.3 단시간 근로자 비례 환산

```
주 소정근로시간 < 40시간 AND ≥ 15시간:
    hoursMultiplier = (주소정근로시간 / 40) × 8
    
    예: 주 24시간 근로자
        hoursMultiplier = (24/40) × 8 = 4.8시간
        연차 15일 = 15 × 4.8 = 72시간 분의 유급휴가
    
    ★ 일수 자체는 통상근로자와 동일하게 15일 부여
    ★ 1일의 가치가 hoursMultiplier(4.8시간)로 환산

법적 근거: 근로기준법 시행령 별표 2
```

## 2. 연차 유효기간 및 소멸

### 2.1 배치(Batch) 단위 관리

```
각 연차는 발생일(grantDate)과 소멸일(expiryDate)을 가진 배치로 관리됩니다.

1년 미만 연차 배치:
    발생일 = 입사일
    소멸일 = 입사 1주년 전날  ← 1년 미만분은 1주년 시점에 소멸

1년 이상 정기 연차 배치:
    발생일 = 입사일 + N년
    소멸일 = 발생일 + 1년 - 1일

법적 근거: 근로기준법 제60조 제7항 (1년간 행사하지 않으면 소멸)
```

### 2.2 FIFO 사용 차감

```
연차 사용 시 오래된 배치부터 우선 차감 (FIFO):

1. 1년 미만 배치 먼저 차감
2. 그 다음 1년차 이후 배치 순서대로 차감

소멸 처리:
    IF 현재일 ≥ 배치 소멸일 AND 미사용분 > 0:
        → 해당 배치의 미사용분 = 소멸
        → 소멸수당 = 미사용일수 × hoursMultiplier × 시급
```

## 3. 연차 사용촉진 (근로기준법 제61조)

### 3.1 촉진 절차

```
1차 통지 기한:
    - 1년 미만 배치: 소멸 3개월 전
    - 정기 배치: 소멸 6개월 전
    
    → 사용자(사장)가 미사용 연차를 서면 통보

2차 통지 기한:
    - 1년 미만 배치: 소멸 1개월 전
    - 정기 배치: 소멸 2개월 전
    
    → 근로자가 사용 시기를 지정하지 않으면 사용자가 지정

촉진 완료 시:
    → 소멸된 연차에 대한 수당 지급 의무 면제 (promotionExemptPayout)
```

## 4. 연차 수당 계산

### 4.1 퇴사 시 잔여 연차 수당

```
IF 퇴사일이 있고 잔여 연차 > 0:
    연차수당 = 잔여 연차 × hoursMultiplier × 시급
    
    예: 잔여 10일, 주 40시간 근로자, 시급 12,000원
        = 10 × 8 × 12,000 = 960,000원

법적 근거: 근로기준법 제60조 제5항
```

### 4.2 1년 미만 미사용 수당

```
1년 미만 발생분 중 미사용분:
    수당 = 미사용일수 × hoursMultiplier × 시급
    
    ★ 이 수당은 1주년 시점에 발생하며 급여 명세서 '기타 수당'에 반영
```

## 5. 기초 연차 (앱 도입 이전)

```
앱 도입 이전에 이미 발생한 연차가 있는 경우:
    initialAdjustment = 사장님이 수동으로 입력한 기초 연차 개수
    initialAdjustmentReason = "앱 도입 이전 3개 잔여" 등 사유

    → totalGenerated에 합산되어 정확한 잔여 연차 산출
```
