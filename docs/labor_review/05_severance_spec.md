# 05. 퇴직금 정산 (Severance Calculator Spec)

> 출처: `packages/shared_logic/lib/src/utils/payroll/severance_calculator.dart`

---

## 1. 퇴직금 지급 조건

```
IF 총 근속일수 ≥ 365일 (1년):
    → 퇴직금 지급 대상
ELSE:
    → 퇴직금 없음

법적 근거: 근로자퇴직급여보장법 제4조
```

---

## 2. 평균임금 산출

### 2.1 기본 공식

```
평균임금 = 퇴직 전 3개월간 총 임금 / 90일

3개월간 총 임금 구성:
  (1) 실제 근로 대가 (시급 × 근무시간)
  (2) 주휴수당 (12.8주 분)
  (3) 연차 미사용 수당 (15일 × 1일소정근로시간 × 시급 × 3/12)

법적 근거: 근로기준법 제2조 제1항 제6호
```

### 2.2 통상임금 하한선

```
통상임금(1일) = hoursMultiplier × 시급

IF 평균임금 < 통상임금:
    → 평균임금 = 통상임금으로 대체

법적 근거: 근로기준법 제2조 제2항
  "평균임금이 통상임금보다 적으면 그 통상임금액을 평균임금으로 한다"
```

### 2.3 데이터 부족 시 처리

```
최근 3개월 실제 출근일이 30일 미만인 경우:
  → 소정근로시간 기반 이론적 3개월 총액 추산
  
  추산 공식:
    주간 유급시간 = 주 소정근로시간 + 주휴시간
    3개월 총액 = 주간 유급시간 × (90 / 7) × 시급

  → 사장님에게 "데이터 부족, 수동 입력 권장" 경고 표시
  → manualAverageDailyWage를 직접 입력하면 그 값 사용
```

### 2.4 단시간 근로자

```
hoursMultiplier = (주소정근로시간 / 40) × 8

예: 주 24시간 근로자
    hoursMultiplier = (24/40) × 8 = 4.8시간
    통상임금(1일) = 4.8 × 10,320 = 49,536원
```

---

## 3. 퇴직금 계산

```
퇴직금 = (평균임금 × 30일) × (총 근속일수 / 365)

예: 평균임금 80,000원/일, 근속 2년(730일)
    = (80,000 × 30) × (730 / 365)
    = 2,400,000 × 2.0
    = 4,800,000원

법적 근거: 근로자퇴직급여보장법 제8조
```

---

## 4. 퇴사 정산 총액

```
총 정산금 = 퇴사월 일할급여 + 잔여 연차수당 + 퇴직금

지급 기한: 퇴사일로부터 14일 이내

법적 근거: 근로기준법 제36조 (금품 청산)
```

### 4.1 퇴사월 일할급여

```
퇴사월 1일 ~ 퇴사일까지의 실제 근무시간 × 시급
(퇴사월 전체 급여를 일할로 산정)
```

### 4.2 잔여 연차 수당

```
잔여 연차 수당 = 잔여 연차일수 × hoursMultiplier × 시급

(annual_leave_calculator에서 산출된 remaining 값 사용)
```

## 5. 결과 항목

| 항목 | 설명 |
|------|------|
| `workerName` | 근로자 성명 |
| `joinDate` | 입사일 |
| `exitDate` | 퇴사일 |
| `totalWorkingDays` | 총 근속일수 |
| `isSeveranceEligible` | 퇴직금 지급 대상 여부 (≥ 365일) |
| `exitMonthWage` | 퇴사월 일할급여 |
| `remainingLeaveDays` | 잔여 연차일수 |
| `annualLeavePayout` | 잔여 연차수당 |
| `severancePay` | 퇴직금 |
| `averageDailyWage` | 평균임금 (1일) |
| `paymentDeadline` | 금품 청산 기한 (퇴사일 + 14일) |
| `requiresManualInput` | 데이터 부족 경고 (수동 입력 권장) |
| `totalSettlementAmount` | 총 정산금 (일할 + 연차 + 퇴직금) |
