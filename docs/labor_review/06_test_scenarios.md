### A-3. 야간근무 포함

| 항목 | 값 |
|------|------|
| 시급 | 11,000원 |
| 근무 | 화~토 18:00~02:00 (8시간, 휴게 1시간) |
| 5인 이상 | YES |

**야간 시간 계산:**
- 18:00~22:00 = 4시간 (일반)
- 22:00~02:00 = 4시간 (야간)
- 휴게 1시간 → 순수 7시간
- 야간 순수 시간 = 4h × (7h/8h) = 3.5시간

**기대 결과 (1일):**
| 항목 | 산식 | 금액 |
|------|------|------|
| 기본급 | 7h × 11,000 | 77,000원 |
| 야간가산 | 3.5h × 11,000 × 0.5 | 19,250원 |
| **1일 합계** | | **96,250원** |

## 시나리오 B: 월급제

### B-1. 기본급 215만 + 식대 20만 + 고정OT 14만 = 총 250만

| 항목 | 값 |
|------|------|
| 기본급 | 2,150,000원 |
| 식대 | 200,000원 (비과세) |
| 고정연장수당 | 140,000원 |
| 주 소정근로 | 40시간 |
| 5인 이상 | YES |
| 정산기간 | 2026-06-01 ~ 2026-06-30 (전월 재직) |

**검증 포인트:**
```
S_Ref = 4.345 × (40 + 8) = 208.56시간
통상시급 = 2,150,000 / 208.56 = 약 10,311원
참고시급 = (2,150,000 + 200,000) / 208.56 = 약 11,270원

최저임금 판정: 10,311원 ≥ 10,320원?
→ Hard Block 경고 발생 가능 (기본급 단독으로 최저임금 미달)
```
